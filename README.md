# krite-collective.github.io

Reading surface for published work by the [Krite Collective](https://krite-collective.github.io) (K.-U. Hess + Claude Opus 4.6).

## Publications

1. **Is Krite an Author?** — An essay on human–machine composite authorship.  
   DOI: [10.5281/zenodo.19162597](https://doi.org/10.5281/zenodo.19162597) · CC BY 4.0

2. **Who Can Rewrite This Paper?** — A cross-model experiment testing whether the authorship argument can be recovered from its output alone.  
   DOI: [10.5281/zenodo.19314164](https://doi.org/10.5281/zenodo.19314164) · CC BY 4.0

3. **Preamble to a Metaphysical Research Programme: Derivation and Architecture** — Five structural requirements for any metaphysical research programme, and the architecture they constrain.  
   DOI: [10.5281/zenodo.19374853](https://doi.org/10.5281/zenodo.19374853) · CC BY 4.0

4. **Detecting Tautology in Human–AI Theory-Building: A Proposal for a Verification Protocol** — An eight-phase protocol for detecting explanatorily empty output in collaborative theory-building.  
   DOI: [10.5281/zenodo.19386827](https://doi.org/10.5281/zenodo.19386827) · CC BY 4.0
